ตัวตนของผม

ผมไม่ใช่คนที่ชอบการตกแต่ง UI หรืองาน Design เท่าไหร่ แต่สิ่งที่ทำให้ผมตื่นเต้นคือ การต่อสายไฟ การเห็น หลอดไฟติดสว่าง จากโค้ดที่เขียน และการสร้างระบบที่ ทำงานได้จริง

💡

สิ่งที่ผมรัก

  • ต่อวงจรแล้ว LED ติด
  • เขียนโค้ดควบคุม Sensor
  • ดูข้อมูลวิ่งบน Serial Monitor
  • แก้ Bug แล้วระบบทำงาน
🔌

ทำไมถึงชอบ IoT

IoT ทำให้โค้ดที่เขียน มีชีวิต ไม่ใช่แค่ตัวอักษรบนจอ แต่มันควบคุมอุปกรณ์จริง ทำให้หลอดไฟติด ทำให้ Buzzer ร้อง ทำให้เซนเซอร์อ่านค่าได้ — มันจับต้องได้ มันมีแสง มันมีเสียง

🛠️

สิ่งที่ผมถนัด

ผมถนัดฝั่ง Hardware + Firmware มากกว่า UI ชอบ debug วงจร ชอบอ่าน datasheet ชอบต่อ breadboard แต่ก็พัฒนา Front-end ได้เพื่อแสดงผลข้อมูลจากอุปกรณ์

⚡ สิ่งที่ทำให้ผมมีความสุข

🔴 สายไฟสีแดง = VCC
⚫ สายไฟสีดำ = GND
🟡 สายไฟสีเหลือง = Signal
💡 LED ที่กระพริบได้
